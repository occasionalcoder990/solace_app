# Companion App

A psychological companion app that creates deep, personalized connections through thoughtful conversations.

## Features

- **Personalized Onboarding** - Understands your communication style and emotional needs
- **Empathetic AI Companion** - Adapts responses based on your personality profile
- **Safe Space** - For thoughts and feelings you can't share with anyone else
- **Conversation Memory** - Builds continuity across sessions

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the app:
   ```
   npm start
   ```

3. Open your browser to: `http://localhost:3000`

## Development

- `npm run dev` - Start with auto-reload using nodemon

## Database

The app uses SQLite database (`companion.db`) to store:
- User profiles and personality data
- Conversation history
- User preferences

Database is created automatically on first run.
